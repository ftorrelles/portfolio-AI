# Portafolio Francisco Torrelles — AI-Augmented Developer

Este es el portafolio profesional de **Francisco Torrelles**, diseñado para destacar como Full-Stack Developer especializado en **AI-Augmented Development** y **LLM Orchestration**.

## 🚀 Filosofía del Proyecto
Este portafolio no es solo una página web; es un showcase técnico construido bajo principios de arquitectura escalable, optimización radical de performance y una experiencia de usuario orientada a la conversión directa.

## 🛠️ Stack Tecnológico
*   **Framework:** Next.js 16+ (App Router)
*   **Lenguaje:** TypeScript
*   **Styling:** Tailwind CSS (v4 pattern)
*   **Internacionalización:** Path-based i18n (SEO-friendly)
*   **Deployment:** Vercel/Netlify

## ✨ Características Principales
*   **AI-Augmented Design:** Código arquitectado para destacar capacidades en IA y Multi-Agent Systems.
*   **SEO-Optimized:** Estructura de rutas internacionalizada (ES/EN) con sitemap dinámico y meta tags para Open Graph.
*   **Performance First:** Optimización de componentes, lazy loading y optimización de assets.
*   **Direct Engagement:** CTA orientado a conversión directa vía WhatsApp.
*   **Design System:** Sistema de diseño oscuro "Nex-Dark" con acentos neón personalizados.

## 🚀 Getting Started

1. **Clonar el repositorio:**
   ```bash
   git clone https://github.com/ftorrelles/portfolio-AI.git
   cd portfolio-AI
   ```

2. **Instalar dependencias:**
   ```bash
   npm install
   ```

3. **Ejecutar en desarrollo:**
   ```bash
   npm run dev
   ```

4. **Construir para producción:**
   ```bash
   npm run build
   npm run start
   ```
